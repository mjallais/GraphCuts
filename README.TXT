Implementation of the method developped in the publication : 
     "Graphcut Textures: Image and Video Synthesis Using Graph Cuts" by Vivek Kwatra

Developpers : Maeliss Jallais, Charly Girot and Maxime DiFolco

Pour compiler notre implémentation : 

- Choisir une texture dans le dossier images disponible. Il faut que le nom du fichier finisse par '_32'.

- Choisir la méthode de placement de patch en faisant varier la varaible "method" : 
     0 correspond au placement de patch par correspondance 
     1 correspond au placement aléatoire
     
- Le résultat se trouva dans le dossier résultats avec le nom de fichier que vous avez choisi

Pour avoir des résultats de plus haute résolution, il faut utiliser les images avec '_128'. Le résultat est alors de meilleur qualité mais nécessite plus de temps de compilation.


Dans le dossier résultat certains résultats détaillé dans le rapport sont disponibles.
